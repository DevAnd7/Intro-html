CONSEJOS:
- El primer archivo debe ser index.html esto se hace asi por convencion, es el archivo que le dice a nuestro explorador que es una pagina web y que debe seguir esa estructura

- <!DOCTYPE html>: indica que es de tipo html5, que es una pagina actual y no de los anios 90

-existen solo 6 heading, del h1 al h6

- <span>: la etiquetq span sirve para referenciar texto

- <a href=""> para hacer links <a target="" href=""> para hacer link en una nueva ventana

- <img src="" />: la etiqueta de imagen se cierra directamente

- atributo 'id': sirve para identificar un elemento para luego darle caracteristicas con JS por ejemplo

- atributo 'name': sirve para identificar el campo en el servidor cuando el formulario fue enviado

-Cuando se envia un formulario tenemos dos metodos GET: para enviar los valores que estan dentro de los campos a traves de la URL. POST: enviaria los valores en las cabeceras y no en el URL

TERMINOS:
-Concatenar: unir variables en una sola linea

-Convencion: algo que se ha hecho asi por conveniencia o costumbre, como un estandar
